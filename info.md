# TechMart UI Style & Color Theory Guidelines

## Design Philosophy

* Modern
* Clean
* Professional
* Consistent across all modules
* Business ERP/Admin Dashboard style

## Color Palette

### Primary

* Blue
* Used for:

  * Main buttons
  * Navbar
  * Active menu
  * Links

### Success

* Green
* Used for:

  * Success messages
  * Active status
  * Save buttons
  * Verified badges

### Warning

* Orange/Yellow
* Used for:

  * Pending status
  * Edit buttons
  * Warning alerts

### Danger

* Red
* Used for:

  * Delete
  * Disable
  * Errors
  * Critical actions

### Secondary

* Gray
* Used for:

  * Cancel buttons
  * Secondary actions
  * Disabled items

## Cards

* White background
* Soft shadow
* Rounded corners
* Consistent spacing

## Buttons

* Primary → Main action
* Success → Save/Create
* Warning → Edit
* Danger → Delete/DisableV
* Secondary → Cancel/Back

## Tables

* Responsive
* Hover effect
* Dark table header
* Status displayed using badges

## Forms

* Labels above inputs
* Consistent spacing
* Validation messages below inputs
* Full-width controls where appropriate

## Icons

* Use Bootstrap Icons
* Pair icons with actions for better usability

## Alerts

* Success → Green
* Error → Red
* Warning → Orange
* Information → Blue

## Navigation

* Sticky top navbar
* Dropdowns for large modules
* Active menu highlighting
* User menu on the right

## UI Principles

* Keep pages uncluttered.
* Reuse components.
* Maintain consistent spacing and typography.
* Avoid changing colors between modules without a reason.
* Prioritize readability and accessibility.
